# Componente: Bot

Este componente tiene una integración con la API de WhatsApp de Twilio.
Recuerde poner las variables de entorno en el archivo .env o en el entorno de ejecución.

```javascript
MICROSOFT_APP_ID=
MICROSOFT_APP_PASSWORD=

TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
TWILIO_PHONE_NUMBER=
TWILIO_ENDPOINT_URL=

BPM_HOST=
BPM_PORT=80
```

Para ejecutar utilice:

```javascript
npm install

npm run start-dev
```
